# Securing-APIs-using-OAuth-2.0-with-GitHub-and-Auth0

# OAuth 2.0 Flask Lab Activity

## 📌 Overview
This project demonstrates OAuth 2.0 authentication using GitHub and Flask. It allows users to log in with GitHub, view a protected profile page, and access a secure API route.

---

## 🚀 Features
- GitHub OAuth Login
- Session-based Authentication
- Protected Profile Route
- Secure API Endpoint
- Logout Functionality

---

## 🛠️ Tech Stack
- Python
- Flask
- Authlib
- GitHub OAuth
